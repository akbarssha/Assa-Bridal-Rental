 💍 ASSA Bride & Groom Rental

🚀 Overview

A full-stack web application that allows brides and grooms to rent wedding outfits online. The platform includes secure user authentication and online payment integration for a seamless booking experience.

 ✨ Features

* 👗 Browse bridal & groom collections
* 🔐 User authentication (Signup/Login)
* 🛒 Outfit booking system
* 💳 Online payments using Razorpay
* 📱 Fully responsive design

🛠️ Tech Stack

(MERN Stack)

* Frontend: HTML, CSS, JavaScript / React
* Backend: Node.js / Express
* Database: MongoDB

 💳 Payment Integration

Integrated Razorpay payment gateway to enable secure and smooth online transactions.

 ⚙️ How to Run Locally

1. Clone the repo
2. Install dependencies
3. Add environment variables (Razorpay keys)
4. Run the server

🎯 Key Learnings

* Implementing secure authentication
* Integrating payment gateway APIs
* Handling real-world user flows
* Managing frontend-backend communication
* Order history page
* Admin dashboard
* Order cancel option
* Product availability tracking
